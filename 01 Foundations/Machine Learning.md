# Machine Learning

Machine Learning (ML) is a subset of [[Artificial Intelligence]] where systems learn patterns from data to make predictions or decisions, without being explicitly programmed for each scenario.

## How It Works

1. **Data collection** — Gather a dataset relevant to the problem
2. **Feature selection** — Identify which attributes of the data are meaningful
3. **Model selection** — Choose an algorithm (e.g., decision tree, neural network)
4. **[[Training and Inference|Training]]** — Feed data through the model, adjusting parameters to minimize error
5. **Evaluation** — Test the model on unseen data to measure performance
6. **[[Training and Inference|Inference]]** — Deploy the model to make predictions on new inputs

## Learning Paradigms

See [[Supervised vs Unsupervised Learning]] for details.

- **Supervised learning** — Learning from labeled examples (input → known output)
- **Unsupervised learning** — Finding structure in unlabeled data
- **[[Reinforcement Learning]]** — Learning through trial, error, and rewards
- **Self-supervised learning** — Generating labels from the data itself (used heavily in [[Large Language Models]])

## Common Algorithms

| Algorithm | Type | Use Case |
|---|---|---|
| Linear Regression | Supervised | Predicting continuous values |
| Logistic Regression | Supervised | Binary classification |
| Decision Trees / Random Forests | Supervised | Classification and regression |
| K-Means | Unsupervised | Clustering |
| SVMs | Supervised | Classification |
| [[Neural Networks]] | Both | Complex pattern recognition |

## Key Challenges

- **Overfitting** — Model memorizes training data instead of generalizing
- **Underfitting** — Model is too simple to capture patterns
- **Bias in data** — Training data reflects societal biases (see [[AI Ethics and Safety]])
- **Interpretability** — Complex models are hard to explain
